Dynamic Compute Management

Dynamic, Automated and Frictionless - with a focus on Cloud Cost Management

cmpute optimizes applications across 4 key parameters - cost, performance, time and availability.

1. Cost

Dynamically manage instances across spot, RI and On-demand and move them across regions, availability zones and instance types based on dynamic market activities to keep the costs low.

2. Performance

Automatically analyze application performance and identify the right instance types for the right application (instance right sizing).

3. Time

Schedule to start and stop instances and applications based on tags and application characteristics.

4. Availability

Automatically balance application workload across instance categories and types based on availability metrics and predicting potential risks of instances getting terminated.
