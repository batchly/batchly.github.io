Why Batchly?

Batchly is a cloud scale data processing service that abstracts AWS infrastructure management complexities and enables businesses to easily run their data / batch processing jobs on AWS for significant time and cost savings.

Batchly's machine learning algorithm identifies the right instance family and size to run the data processing workload and optimizes for cost and time (using AWS spot instances). With connectors to AWS based data services, enterprises can start using Batchly and AWS without any change to their existing code / batch process.

Features include hands free processing, outstanding security and an interactive dashboard. Use Batchly for image processing, media transcoding, document processing and any large enterprise data processing jobs.

Why Batchly Benefits You

Batchly intelligently uses AWS EC2 Spot instances for interrupt tolerant workloads and saves upto 90% on AWS costs.

It also has a continous learning algorithm which understands the workload and does the right instance sizing. For. e.g. a CPU intensive workload would be run on c3 family or a memory intensive workload would run on m3 family of EC2 instances. This right sizing helps your application run more efficiently and cheaply. The more you run it and better it gets in terms of cost and time.

Batchly provides complete abstraction on cloud infrastructure management. It has a self healing mechanism to handle errors and maintain application state.

For a DevOps engineer, Batchly gives peace of mind. He/She can easily integrate their applications with Batchly via APIs. It is completely secure and runs inside customer's VPC and provides complete automation - from instance sizing to spot provisioning and management.

For a business, Batchly saves upto 90% on AWS costs. It is frictionless to integrate with existing systems with little to no IT overhead. Batchly also provides comprehensive reporting & control giving full visibility into their cloud costs.
